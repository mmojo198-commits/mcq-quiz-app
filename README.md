# MCQ Quiz Application

An interactive multiple-choice quiz application built with Streamlit.

## Features
- Upload quiz questions from Excel files
- Customizable timer per question
- Question shuffling option
- Real-time scoring and feedback
- Progress tracking with question map

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `streamlit run app.py`
3. Upload an Excel file with columns: Question, Option A, Option B, Option C, Option D, Correct Answer, Hint
